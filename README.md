# SDG-Hyprland-Additions

In this repository, you'll find some of the hyprland additions i've made to my own system, repackaged for public use. 

Each directory contains all the components you need for one addition, generally this is some scripts, configuration snippets for the relevant programs and an install.md file that tells you how to actually configure your system with the service. 

by default, all the paths are set up for you to clone this repository directly into your /home/USERNAME folder. Cloning somewhere else or moving the scripts is totally supported, but will require additional modification.

you can find the list of additions in index.md
